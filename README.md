# 🎬 Slide Animation Website

A simple and visually engaging **image slideshow animation project** built using **HTML, CSS, and JavaScript**. This project showcases smooth transitions and animated slides, making it ideal for beginners learning frontend development.

---

## 🚀 Live Demo

🔗 https://SureshKumar333-dot.github.io/Slide-animation/

---

## 📌 Features

* 🎞️ Smooth image slideshow animation
* 🎨 Clean and minimal UI design
* ⚡ Fast and lightweight (no frameworks)
* 📱 Beginner-friendly project structure
* 🔄 Automatic slide transitions

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Animations & Transitions)
* JavaScript (Basic DOM manipulation)
